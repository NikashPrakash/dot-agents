# Advisory Board

Review the skill from three angles:

1. **Docs maintainer**
   Check whether the workflow will keep `docs/PLATFORM_DIRS_DOCS.md` accurate and easy to maintain over repeated refreshes.

2. **Source-of-truth reviewer**
   Check whether the skill clearly prefers official vendor docs, distinguishes inference from documented fact, and avoids stale-link drift.

3. **Context-efficiency reviewer**
   Check whether the skill uses progressive disclosure, keeps `SKILL.md` lean, and pushes detailed rules into `instructions/`.
